# Support

ParaCalc369 is a small public utility project. The best way to ask for help is to open a GitHub issue.

## Before opening an issue

Please check:

- The README for setup and deployment notes.
- The roadmap for planned work.
- Existing issues, if available.

## Helpful details

For a bug report, include:

- What you were trying to do.
- What happened instead.
- Browser and device.
- Steps to reproduce.
- Screenshot or console output if useful.

For a feature request, include:

- The problem it solves.
- A simple example.
- Which part of the app it affects.
