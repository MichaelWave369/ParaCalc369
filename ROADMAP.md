# ParaCalc369 Roadmap

ParaCalc369 is a free scientific, graphing, unit conversion, formula-reference, installable, and shareable calculator.

## Current foundation

- Scientific calculator engine without unsafe `eval`.
- Graph mode with canvas rendering.
- Unit converter mode.
- Formula Library mode.
- History Vault mode.
- Shareable workspace URLs.
- PWA install/offline shell.
- Unified Tools Dock.
- GitHub Pages deployment pipeline.

## Near-term roadmap

### v2.2 Calculator UX polish

- Enter-to-solve from the main expression input.
- Better parser error messages.
- More visible angle-mode context for graphing.
- Cleaner mobile spacing around the Tools Dock.

### v2.3 Graph upgrades

- Manual y-min/y-max controls.
- Better axis labels and tick marks.
- Export graph with formula and range caption.

### v2.4 Formula Library upgrades

- Insert formula examples directly into calculator.
- Add algebra, trigonometry, statistics, and engineering sections.
- Add formula tags.
- Add notes fields where needed.

### v2.5 Unit converter upgrades

- More engineering units.
- Compound units.
- Custom quick conversions.
- Copy full conversion sentence.

### v3.0 Education Mode

- Guided examples.
- Classroom-friendly worksheet exports.
- Clear learning notes.

## Quality goals

- Keep it fast.
- Keep it installable.
- Keep it understandable.
- Keep it free.
- Keep the core calculator test-covered.
